## 组件规范

### 评分卡片
- 大号数字（48px）+ 进度环/进度条
- 颜色：≥90 绿色，70-89 橙色，<70 红色
- 悬浮：shadow-md，上移 2px

### 营养进度条
- 高度 8px，圆角 4px
- 背景 oklch(0.94)，填充使用品牌色
- 动画：宽度从 0 过渡到目标值，600ms ease-out

### 食谱卡片
- 餐次标题 + 菜品列表 + 热量标签
- 卡片间距 1rem，内部 padding 1.25rem
- 边框左侧 4px 餐次色条（早餐-橙、午餐-绿、晚餐-蓝）

### 数据表格
- 表头固定，行 hover 高亮
- 状态列使用彩色标签（badge）
- 支持 responsive 横向滚动
