# 食材分类标准

## 10大食材主类

### 1. 谷薯类
提供碳水化合物、膳食纤维、B族维生素

**代表性食材：**
- 大米、糯米、黑米、小米
- 面粉、面条、馒头、面包
- 玉米、燕麦、荞麦
- 红薯、紫薯、土豆、山药、芋头

### 2. 蔬菜类
提供维生素、矿物质、膳食纤维

**代表性食材：**
- **叶菜类：** 菠菜、小白菜、油菜、生菜、芹菜、韭菜
- **根茎类：** 胡萝卜、白萝卜、莲藕、洋葱
- **瓜果类：** 黄瓜、冬瓜、南瓜、丝瓜、苦瓜
- **茄果类：** 番茄、茄子、辣椒
- **豆类蔬菜：** 四季豆、豇豆、毛豆
- **菌菇类：** 香菇、金针菇、木耳、银耳

### 3. 水果类
提供维生素、矿物质、抗氧化物质

**代表性食材：**
- 苹果、梨、香蕉、橙子
- 葡萄、草莓、蓝莓、猕猴桃
- 西瓜、哈密瓜、火龙果
- 枣、桃、李子、杏

### 4. 畜禽肉类
提供优质蛋白质、铁、锌、维生素B12

**代表性食材：**
- **畜肉：** 猪肉（瘦肉、排骨）、牛肉、羊肉
- **禽肉：** 鸡肉、鸭肉、鹅肉
- **内脏：** 猪肝、鸡肝、猪血

### 5. 水产类
提供优质蛋白质、DHA、碘、锌

**代表性食材：**
- **鱼类：** 草鱼、鲫鱼、鲤鱼、带鱼、鲈鱼、黄鱼
- **虾蟹类：** 基围虾、河虾、螃蟹
- **贝类：** 扇贝、蛤蜊、牡蛎
- **其他：** 鱿鱼、墨鱼、海带、紫菜

### 6. 蛋类
提供优质蛋白质、卵磷脂、维生素A、D

**代表性食材：**
- 鸡蛋、鸭蛋、鹌鹑蛋、鸽子蛋

### 7. 奶类
提供钙、优质蛋白质、维生素D

**代表性食材：**
- 牛奶、酸奶、奶酪、奶粉

### 8. 豆制品
提供植物蛋白、钙、大豆异黄酮

**代表性食材：**
- 豆腐、豆干、腐竹、豆浆
- 黄豆、绿豆、红豆、黑豆

### 9. 坚果类
提供不饱和脂肪酸、维生素E、矿物质

**代表性食材：**
- 花生、核桃、杏仁、腰果
- 开心果、榛子、松子

### 10. 油脂调味类
提供能量、必需脂肪酸、风味

**代表性食材：**
- 植物油（花生油、大豆油、玉米油、橄榄油）
- 动物油（猪油、黄油）
- 盐、糖、酱油、醋、料酒

## 食材搭配禁忌

### 不宜同食的组合
1. 菠菜 + 豆腐：草酸与钙结合影响吸收
2. 萝卜 + 人参：药性相冲
3. 螃蟹 + 柿子：易形成胃结石
4. 牛奶 + 巧克力：钙与草酸结合

### 推荐搭配
1. 维生素C + 铁：促进铁吸收（如：青椒炒肉）
2. 钙 + 维生素D：促进钙吸收（如：牛奶晒太阳）
3. 蛋白质 + 碳水：提高蛋白质利用率

## 食材选择原则

### 优先选择
- 新鲜、当季、本地食材
- 深色蔬菜（占蔬菜总量50%以上）
- 全谷物（占谷物总量30%以上）
- 优质蛋白质（鱼、禽、蛋、奶、豆）

### 适量控制
- 高盐食材：腌制食品、加工肉类
- 高糖食材：甜点、含糖饮料
- 高脂食材：油炸食品、肥肉

### 避免使用
- 过期变质食材
- 来源不明食材
- 高风险食材（野生蘑菇、河豚等）
