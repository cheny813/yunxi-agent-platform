## 调整原则

### 根据活动量调整
- 高活动量学生：增加主食和肉类10-20%
- 低活动量学生：减少主食和肉类5-10%

### 根据季节调整
- 夏季：增加清淡蔬菜、减少油腻肉类
- 冬季：增加高热量食物、增加根茎类蔬菜

### 根据特殊需求调整
- 体育特长生：增加蛋白质和碳水化合物
- 肥胖学生：增加蔬菜、减少主食和油脂
- 贫血学生：增加红肉和富含维C的蔬菜

## 注意事项

1. 以上分量均为生重，需根据食材特性折算熟重
2. 实际操作中可根据学生反馈适当调整
3. 保证食材新鲜，避免隔夜剩菜
4. 注意食物过敏问题，提供替代方案
