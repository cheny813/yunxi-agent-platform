# 评分规则详解

## 评分维度

| 维度 | 权重 | 达标要求 |
|------|------|---------|
| 能量供给 | 30% | 热量在推荐量 80%-120% |
| 宏量营养素 | 25% | 蛋白质12-15%、脂肪20-30%、碳水55-65% |
| 微量营养素 | 20% | 钙/铁/锌/维A/B1/B2/C 达标 |
| 食材多样性 | 15% | 每餐>=5种，全天>=12种，覆盖8大类 |
| 餐次结构 | 10% | 每餐有主食+蛋白质+蔬菜 |

## 能量供给评分 (30分)

### 热量达标率计算
```
达标率 = 实际热量 / 推荐热量 * 100%
```

### 评分标准
| 达标率范围 | 得分 |
|-----------|------|
| 95%-105% | 30分 (满分) |
| 90%-110% | 25分 |
| 80%-120% | 20分 |
| 70%-130% | 10分 |
| 其他 | 0分 |

## 宏量营养素评分 (25分)

### 理想供能比
| 营养素 | 理想比例 | 允许范围 |
|--------|---------|---------|
| 蛋白质 | 12-15% | 10-18% |
| 脂肪 | 20-30% | 18-35% |
| 碳水化合物 | 55-65% | 50-70% |

### 评分细则
- 三大营养素全部在理想范围内：25分
- 在允许范围内：每偏离理想范围1%，扣1分
- 超出允许范围：该项0分

## 微量营养素评分 (20分)

### 必达营养素
| 营养素 | 重要性 | 常见缺乏症状 |
|--------|--------|-------------|
| 钙 | 高 | 骨骼发育不良 |
| 铁 | 高 | 贫血、注意力不集中 |
| 锌 | 中 | 食欲不振、生长迟缓 |
| 维生素A | 高 | 视力下降、免疫力降低 |
| 维生素B1 | 中 | 疲劳、食欲不振 |
| 维生素B2 | 中 | 口角炎、皮炎 |
| 维生素C | 高 | 免疫力降低、牙龈出血 |

### 评分计算
```
得分 = (达标营养素数量 / 总营养素数量) * 20
```

## 食材多样性评分 (15分)

### 食材种类要求
- 每餐至少5种不同食材
- 全天至少12种不同食材
- 覆盖8大食材主类

### 8大食材主类
1. 谷薯类
2. 蔬菜类
3. 水果类
4. 畜禽肉类
5. 水产类
6. 蛋类
7. 奶类
8. 豆制品

### 评分细则
| 覆盖类别数 | 得分 |
|-----------|------|
| 8类 | 15分 |
| 7类 | 12分 |
| 6类 | 9分 |
| 5类 | 6分 |
| <5类 | 0分 |

## 餐次结构评分 (10分)

### 每餐结构要求
- 主食：提供碳水化合物和能量
- 蛋白质菜：提供优质蛋白质
- 蔬菜：提供维生素和膳食纤维

### 评分标准
- 每餐结构完整：10分
- 缺少一项：扣3分
- 缺少多项：0分

## 综合评分计算

```
总分 = 能量得分 + 宏量营养素得分 + 微量营养素得分 + 多样性得分 + 结构得分
```

### 评分等级
| 总分 | 等级 | 说明 |
|------|------|------|
| >= 90 | 优秀 | 营养均衡，值得推广 |
| 80-89 | 良好 | 基本达标，可适当优化 |
| 70-79 | 合格 | 需要改进 |
| < 70 | 不合格 | 需要重新规划 |
