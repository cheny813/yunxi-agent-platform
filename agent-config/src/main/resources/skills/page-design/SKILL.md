# Page Design Skill

## Overview

This skill enables AI-assisted web page design and generation. It provides design guidelines, template recommendations, and best practices for creating professional, modern web interfaces.

## Design Principles

### Visual Design
- Use a cohesive color palette (primary + accent + neutral)
- Maintain consistent spacing (4px/8px grid system)
- Apply typography hierarchy (H1-H6 with clear size differences)
- Ensure sufficient contrast ratios for accessibility (WCAG AA minimum)

### Layout Guidelines
- Use responsive grid systems (12-column or flex-based)
- Maintain visual hierarchy through size, color, and whitespace
- Group related elements logically
- Ensure mobile-first responsive design

### Component Patterns
- Cards for content grouping
- Tables for tabular data with sorting/filtering
- Charts (bar/line/pie) for data visualization
- Forms with clear validation feedback
- Navigation with breadcrumbs and clear CTAs

## Template Types

### Dashboard
- KPI cards at top (4-column grid)
- Charts section (2-column: line + bar)
- Data table with pagination
- Sidebar navigation

### List/Table
- Search bar + filter row
- Sortable columns
- Pagination controls
- Batch action toolbar

### Report
- Header with title and date range
- Summary statistics section
- Chart section (mixed chart types)
- Detailed data tables
- Export buttons (PDF/Excel)

### Form
- Multi-column layout for related fields
- Validation indicators
- Progress indicator for multi-step forms
- Clear submit/cancel actions

## Color Guidelines
- Primary: brand color (60% usage)
- Accent: action/highlight color (30% usage)
- Neutral: background/text (10% usage)
- Success: green for positive states
- Warning: amber for caution states
- Error: red for error states

## Accessibility Checklist
- All images have alt text
- Form fields have labels
- Color is not the only indicator
- Focus states are visible
- Keyboard navigation works
- Minimum contrast ratio 4.5:1 for text
