# 13. 更新日志

## [1.0.0] - 2026-05-09

### 新增

- 初始版本发布
- 支持多 Agent 协作
- 集成 MCP 协议
- 规则引擎支持
- 多平台接入（Web、企业微信、钉钉、飞书）

### 模块

- agent-core: 核心框架
- agent-gateway: 统一网关
- agent-rule-engine: 规则引擎
- agent-business: 业务实现
- agent-text2sql: SQL 生成
- mcp-common: MCP 协议
